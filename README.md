# Text-classification on Keras Tensorflow with LSTM and word2vector models.

Prepare python:

```shell
pip install -r ./requirements.txt
```

Train model:
```shell
python ./model.py
```

Evaluate saved model:
```shell
python ./eval.py
```

Run Tensorboard
```
make tensorboard
```
http://localhost:6006/#scalars
